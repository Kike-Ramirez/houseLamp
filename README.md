# >> houseLamp v0.1 <<

## Description
Arduino sketch controlling a LED Strip in a domestic environment. It acts as a web server in a LAN offering the user to control the LED Lamp remotely.

## Hardware Required
- Arduino UNO Rev.3
- Ethernet Shield
- Digital Addressable LED Strip 12V
- Power Supply 12V
